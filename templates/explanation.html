SHAP Explanation

Understanding how the Random Forest model makes dropout predictions using SHapley Additive exPlanations

What is SHAP?

SHapley Additive exPlanations (SHAP) is an Explainable AI (XAI) method based on game theory's Shapley values. It explains each prediction by computing how much each feature contributed to the final outcome.

Positive SHAP Value

This feature INCREASES the dropout risk prediction. Example: Low grades push the prediction toward "At-Risk."

Negative SHAP Value

This feature DECREASES the dropout risk prediction. Example: High grades push the prediction toward "Not At-Risk."

How It Works
  1. Model makes a prediction for a student
  2. SHAP computes each feature's contribution
  3. Positive values = increase risk
  4. Negative values = decrease risk
  5. Sum of all SHAP values = final prediction

Feature Category Contribution
Feature Category Breakdown
Academic Features (Strongest Impact)

Grade 1, Grade 2, Final Grade, Number of Failures, Study Time. Academic performance is the most significant predictor. Students with declining grades and past failures are significantly more likely to drop out.

Attendance Features

Number of Absences. High absenteeism is a strong indicator of disengagement and potential dropout.

Behavioral Features

Free Time, Going Out, Alcohol Consumption, Health Status, Extra-Curricular Activities. Behavioral patterns provide additional context to academic indicators.

Demographic Features

Age, Gender, Parental Education, Family Size, Address. Background factors contribute to the overall risk profile but are less influential than academic factors.
